Description

Como executar os notebooks:
  - clonar
  - cd P2019.1_PAA_SEM1
  - virtualenv venv (primeira vez)
  - source activate
  - pip install -r requirements.txt (primeira vez)
  - python setup.py develop (primeira vez)
  - cd notebooks
  - jupyter-notebook
